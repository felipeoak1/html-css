Usando links externos

Utilizamos a âncora que é a tag <a>

Também podemos utilizar o parâmetro rel, onde ele indica qual o destino do nosso link, algumas opções que temos são:

  • next - Indica que o link é para a próxima parte do documento.
  • prev - Indica que o link é para a parte anterior do documento atual.
  • author - Indica que é um link para o site do autor do artigo atual.
  • external - Indica que é um link para outro site que não faz parte do site atual.
  • nofollow - Indica que é um link para um site não endossado, como um link pago.

E o parâmetro href indicamos o link. Também podemos utilizar o parâmetro target com as seguintes opções:

  • _blank - Abre uma nova guia com o link.
  • _self - Abre o site sobrepondo o nosso site (valor padrão).

Outra opção também é quando queremos indicar um link para download, para isso, referênciaremos o endereço do arquivo no parâmetro href, utilizaremos o parâmetro "download=" com o nome do arquivo que estamos disponibilizando e por último, o parâmeto "type", para indicarmos o tipo do nosso arquivo. Para mais informações sobre os tipos de arquivo, consulte o site oficial da IANA
